# Car ROI Calculator

Standalone SPA — калькулятор доходности сдачи автомобиля в аренду для инвесторов.

## Стек технологий

| Инструмент | Версия | Причина выбора |
|---|---|---|
| **React + TypeScript** | 19 / 5 | Типобезопасность, компонентный подход, экосистема |
| **Vite** | 8 | Мгновенный HMR, быстрая сборка, нет оверхеда Next.js (SSR не нужен) |
| **Tailwind CSS v4** | 4 | Утилитарные классы, нулевой runtime, отличная адаптивность |
| **Recharts** | 2 | Декларативные графики на SVG, хорошо интегрируется с React |
| **jsPDF** | 3 | Генерация PDF прямо в браузере, без серверной части |

## Архитектура

```
src/
├── components/
│   ├── InputForm.tsx      # Форма ввода параметров
│   ├── ResultsPanel.tsx   # KPI-карточки + разбивка доходов
│   └── RevenueChart.tsx   # Графики (структура / накопленная прибыль)
├── hooks/
│   └── useCalculator.ts   # Состояние + useMemo для пересчёта
├── types/
│   └── index.ts           # CarInputs, CalcResults
├── utils/
│   ├── calculations.ts    # Чистые функции расчёта (без побочных эффектов)
│   ├── format.ts          # Форматирование валюты, процентов, месяцев
│   └── generatePdf.ts     # Генерация PDF-отчёта через jsPDF
└── App.tsx                # Корневой компонент, layout
```

**Принципы:**
- Логика расчётов полностью отделена от UI (`utils/calculations.ts`)
- Хук `useCalculator` — единственное место состояния
- Компоненты — чистые presentational, получают данные через props

## Формулы расчёта

```
Месячный оборот       = цена аренды × дни аренды
Комиссия              = оборот × (комиссия% / 100)
Доход после комиссии  = оборот − комиссия
Чистая прибыль        = доход после комиссии − расходы
ROI (годовой)         = (чистая прибыль × 12) / стоимость авто × 100
Окупаемость (мес.)    = стоимость авто / чистая прибыль
```

## Запуск локально

```bash
git clone <repo-url>
cd car-roi-calculator
npm install
npm run dev
```

Открыть `http://localhost:5173`

### Сборка для продакшена

```bash
npm run build
# артефакты в папке dist/
```
